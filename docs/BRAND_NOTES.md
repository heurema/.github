# Heurema Brand Notes

## Core identity

Name: `heurema`

Style: lowercase in code/open-source contexts, `Heurema` in business prose.

Meaning: discovery, finding, invention, useful result.

Taglines:

```text
Craft, not conjuring.
From business intent to verified code change.
Bounded, auditable AI-agent systems for software delivery.
```

## Voice

Heurema should sound:

- engineering-first;
- skeptical of magic claims;
- practical;
- security-aware;
- vendor-neutral;
- calm and precise.

It should not sound:

- like a generic AI agency;
- like a tool reseller;
- anti-vendor;
- overpromising;
- enterprise-bureaucratic.

## Logo concept

The mark combines three ideas:

1. `h` — Heurema.
2. rails/gates — bounded delivery.
3. a check node — verified change.

Preferred usage:

- GitHub avatar: `assets/heurema-avatar.png`
- GitHub README header: `assets/heurema-wordmark.svg`
- Docs/favicon: `assets/heurema-mark.svg`

## Color direction

Keep the palette calm and technical:

- near-black background;
- off-white text;
- one accent color for verified/gate signal.

Do not use flashy “AI gradient magic” branding. That would conflict with `Craft, not conjuring`.
