# Commercial Entry Point

## First public offer

```text
Heurema AI Delivery & Security Audit
```

## Promise

In 5 business days, identify where AI agents can safely improve software delivery without vendor lock-in, security debt, or fake ROI claims.

## Best first ICP

- software/product agencies with 10–50 people;
- SaaS/product teams with 5–30 engineers;
- teams already using AI tools informally;
- teams with delivery bottlenecks in requirements, context, review, tests, documentation, or legacy onboarding.

## Deliverables

1. Current SDLC map.
2. AI readiness score.
3. Security/data risk map.
4. Agent workflow opportunity backlog.
5. Vendor lock-in risk map.
6. First pilot workflow.
7. Agent permission matrix.
8. ROI baseline.
9. 30-day implementation roadmap.

## First pilot

```text
Goalrail Pilot: from business intent to verified code change
```

Scope one workflow only:

- business intent → delivery contract → context package → agent-assisted PR → tests → review → proof.

## What not to sell first

- full AI transformation;
- autonomous production deployment;
- broad custom development;
- “replace developers” promises;
- model/platform resale;
- heavily regulated clients without a mature playbook.
