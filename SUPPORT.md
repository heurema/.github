# Support

Heurema is open-source-first. Support depends on the repository maturity level.

## For bugs

Open a GitHub issue with:

- repository name;
- version or commit;
- operating system;
- expected behavior;
- actual behavior;
- reproduction steps;
- logs with secrets removed.

## For security issues

Do not open a public issue. Follow `SECURITY.md`.

## For implementation help

For teams adopting AI-agent workflows, the recommended starting point is a bounded audit:

```text
AI Delivery & Security Audit
```

The goal is to map the current SDLC, identify safe first workflows, define agent permissions, and avoid vendor lock-in.
